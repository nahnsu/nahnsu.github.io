2 Letters 1 Shape Project

This was a personal project to build a tool for generating 3d models of objects that look like one letter from one angle, and a different letter from a different angle. Here, maybe it's best if I just show you - like this:

It was originally inspired by a cute present idea I had for my boyfriend (since we have the same number of letters in our name), to make something that looked like my name from one angle, and his from another. Of course, I immediately started scheming as to how to make this completely generalizable, as in the curse of every mathematician.

Right now, this tool has to be run locally, and the code for it lives here.

I also wrote a blog post about using different generative AI models (OpenAI, Grok, and Claude) to define the geometric points of each letter, which can be found here
